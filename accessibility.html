ウェブアクセシビリティ方針

1. 基本方針

BUSON STUDIO（以下「当ブランド」といいます）は、すべての人がウェブサイトを等しく利用できるよう、ウェブアクセシビリティの確保・向上に取り組んでいます。年齢や障害の有無に関わらず、多様なユーザーがウェブコンテンツにアクセスし、理解し、操作できることを目指しています。

当サイトは、JIS X 8341-3:2016「高齢者・障害者等配慮設計指針－情報通信における機器，ソフトウェア及びサービス－第3部：ウェブコンテンツ」および国際規格であるWCAG 2.1（Web Content Accessibility Guidelines）のAA準拠を目標として取り組んでいます。

2. アクセシビリティ対応の主な取り組み

当サイトでは、以下のようなアクセシビリティ対応を行っています：

テキスト

読みやすいフォント、適切な文字サイズと行間、十分な色のコントラストを確保

レスポンシブデザイン

スマートフォンやタブレットなど様々な画面サイズに対応

キーボード操作

マウスを使わずにキーボードだけで全ての機能を利用可能

代替テキスト

画像には適切な代替テキストを提供

スクリーンリーダー

スクリーンリーダーでの読み上げに配慮した構造

適切なマークアップ

HTML5の意味的なマークアップを使用

これらの取り組みにより、視覚・聴覚・身体・認知などの障害がある方や、高齢の方でも利用しやすいウェブサイトの実現を目指しています。

3. 対応ブラウザと推奨環境

当サイトは以下のブラウザの最新版での利用を推奨しています：

Google Chrome
Mozilla Firefox
Safari
Microsoft Edge

旧バージョンのブラウザやサポートが終了したブラウザでは、一部機能が正常に動作しない場合があります。セキュリティの観点からも、ブラウザは常に最新の状態に保つことをお勧めします。

4. アクセシビリティ対応の限界

当サイトでは可能な限りアクセシビリティに配慮していますが、以下の点については現時点で完全な対応ができていない場合があります：

  • YouTubeなどの外部サービスから埋め込まれたコンテンツ
  • アニメーションや動きのあるコンテンツの一部
  • 過去のアーカイブコンテンツの一部

これらの課題については、今後も改善に取り組んでまいります。

5. 支援技術対応

当サイトは、以下の支援技術と組み合わせての利用を考慮しています：

  • スクリーンリーダー（NVDA、VoiceOver、JAWS など）
  • 画面拡大ソフト
  • 音声認識ソフトウェア
  • キーボードナビゲーション
  • 読み上げ機能

これらの支援技術を使用する場合も、最新のブラウザとの組み合わせでの利用を推奨します。

6. ユーザー補助機能

ブラウザやオペレーティングシステムに組み込まれている以下のユーザー補助機能も活用できます：

  • ブラウザの文字サイズ変更機能：Ctrl + プラス/マイナスキー（Windows/Linux）、Command + プラス/マイナスキー（Mac）
  • ブラウザのズーム機能
  • ハイコントラストモード（オペレーティングシステムの設定から変更可能）
  • キーボードナビゲーション（Tabキーでのフォーカス移動）

7. 継続的な改善

ウェブアクセシビリティは継続的に改善すべき課題と認識しています。当サイトでは、以下の取り組みを行っています：

  • 定期的なアクセシビリティ監査とテスト
  • ユーザーフィードバックの収集と分析
  • アクセシビリティ関連の最新技術や基準への対応
  • サイト制作・更新に関わるスタッフへのアクセシビリティ教育
